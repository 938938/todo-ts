## 사용방법
1. 새로운 할 일 입력창에 입력하시고 키보드의 Enter를 누르시거나 추가 버튼을 누르시면 할 일이 입력됩니다.
2. 중요한 일은 커서로 잡아서 '중요한 일'로 옮겨주세요.
3. 완료한 할 일은 클릭하시면 '완료한 일' 목록으로 이동됩니다.
4. 각 목록은 클릭하면 열고 닫을 수 있습니다.
